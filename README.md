# Python Teaching Material for Beginners

A collection of beginner-friendly Python tutorials covering the core language concepts with clear examples and explanations.

## Topics Covered

| File | Topic |
|------|-------|
| [01_hello_world.py](01_hello_world.py) | Hello World and `print()` |
| [02_variables_and_types.py](02_variables_and_types.py) | Variables and data types |
| [03_strings.py](03_strings.py) | String operations and methods |
| [04_numbers.py](04_numbers.py) | Numbers and arithmetic |
| [05_lists.py](05_lists.py) | Lists and list operations |
| [06_tuples.py](06_tuples.py) | Tuples |
| [07_dictionaries.py](07_dictionaries.py) | Dictionaries |
| [08_sets.py](08_sets.py) | Sets |
| [09_conditionals.py](09_conditionals.py) | Conditional statements (`if`/`elif`/`else`) |
| [10_loops.py](10_loops.py) | `for` and `while` loops |
| [11_functions.py](11_functions.py) | Functions |
| [12_modules.py](12_modules.py) | Modules and imports |
| [13_exceptions.py](13_exceptions.py) | Exception handling |
| [14_classes.py](14_classes.py) | Object-oriented programming basics |

## How to Use

Each file is a standalone Python script. Run any file directly with Python 3:

```bash
python 01_hello_world.py
```

Follow the files in order for a guided learning experience, or jump to any topic you need.

## Prerequisites

- Python 3.x installed ([python.org](https://www.python.org/downloads/))
